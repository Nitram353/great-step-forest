🔐 Admin access

Enter the admin passcode to manage competition data.

⚙️ Publishing setup

Changes are published by committing directly to the site's GitHub repository. Paste a GitHub personal access token with write access to the repo — it is stored only in this browser (localStorage), never on the site.

🔗 Sign-up & donation links

The buttons on the home page point to these. Leave a field empty until the page exists — its button will show "coming soon" until then. (Event sign-up links are set per event in the Events section below.)

📣 Weekly announcement

Shown in the "Race Across the World — weekly update" panel at the top of the home page. Update it each week to say which destination each team has reached. Leave empty to hide the panel.

🚶 Walkers

Number of colleagues walking in each team. The "colleagues walking" tile on the home page adds these up — or type a manual total below to override the sum.

👟 Weekly steps

Enter total steps recorded by each team, per week. Leave future weeks at 0.

The multiplier scales every kilometre shown on the site (leaderboards and the Race Across the World map). At 1.0 distances are real; raise it (e.g. 5) if teams need a boost to reach the later destinations.

💚 Money raised

Running fundraising total per team, in pounds.

⭐ Top individual steppers

The top 5 shown on the leaderboard (cumulative steps). First name + initial is plenty. This section is currently hidden on the site — tick the box to show it if you find a way to identify top steppers.

📌 Events

Events shown in the sidebar. Mark one as “featured” to pin it to the top in the big blue card.

🚀 Publish

Commits your changes to GitHub. The live site updates within a minute or two.
